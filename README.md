# realistic-quantum-network-simulation
This repo contains code and data to reproduce results in the paper "Realistic quantum network simulation for BBM92 key distribution". This paper uses the Aliro Simulator to simulate BBM92 key distribution, reproducing experimental results and verifying analytical theory.

## Running the code

The theory plots in Figures 2 and 3 (predicting BBM92 key rates and QBERs) can be generated by running the bash script `run_theory.sh`. The results will be stored in json files with the data type "numerics" provided within the file.

The Aliro Simulator (AQNSim) plots in Figures 2 and 3 (simulating BBM92 key rates and QBERs) can be generated by running the bash script `run_simulations.sh`. The results will be stored in json files with the data type "aqnsim" provided within the file.

The experiment plots in Figures 2 and 3 (calculating BBM92 key rates and QBERs) can be generated from the raw saved data by running the bash script `run_exp_analysis.sh`. The results will be stored in json files with the data type "experiment" provided within the file. Running this bash script will also generate Figures 6 and 7 from the raw saved data.

The simulations and plots for Figures 4 and 5 can be generated by running the bash script `run_repeater_chains.sh`. This script will run Aliro Simulator simulations as well as calculate theory.

To create Figures 2 and 3 from the theory, Aliro Simulator, and experimental analysis files, run the bash script `run_make_comparison_plot.sh`.

Raw experimental data is saved in the folder `timetagged_data`.
